GIT alapvető parancsok

git init: Új Git repository inicializálása az aktuális mappában.

git status: Megmutatja a fájlok aktuális állapotát a munkakönyvtárban.

git add: A módosított fájlokat a staging area-ba helyezi.

git commit: Elmenti a staging area tartalmát egy verzióként.

git branch: Listázza vagy létrehozza az ágakat.

git checkout: Átvált egy másik ágra vagy commitra.

git push: Feltölti a commitokat a távoli repository-ba.

git pull: Letölti és egyesíti a távoli változásokat.
