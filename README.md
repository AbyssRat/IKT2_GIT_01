 # Webes feladat összefoglaló

A feladat célja egy egyszerű webes projekt elkészítése, amely bemutatja az alapvető webfejlesztési technológiák használatát. A projekt során strukturált weboldal készül, verziókezeléssel és alap stílusozással, átlátható felépítésben.

## Felhasznált technológiák

* GIT
* HTML
* CSS3
* JavaScript

**GIT**: Verziókezelő rendszer, amely lehetővé teszi a projekt változásainak követését, visszaállítását és a csapatmunkát. Segít abban, hogy a fejlesztés biztonságos és jól dokumentált legyen.

**HTML**: Egy leíró nyelv, amelyet a böngésző értelmez, és a benne található elemek alapján megjelenít egy weboldalt. A HTML struktúrát ad az oldalnak különböző tagek (<tagnév></tagnév>) segítségével.

**CSS3**: Stílusleíró nyelv, amely a HTML elemek megjelenéséért felel. Segítségével beállíthatók a színek, betűtípusok, elrendezések és animációk, így az oldal esztétikusabbá válik.

**JavaScript**: Programozási nyelv, amely a weboldal működését és interaktivitását biztosítja. Lehetővé teszi az eseménykezelést, dinamikus tartalomkezelést és a felhasználói élmény javítását.
